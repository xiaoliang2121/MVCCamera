现在碰到一个问题：相机只在序列图开始的时候接收到一次触发信号，在dlp4500投影仪显示接下来的图片时没有获得触发。而需要的是在每张图案显示时触发一次，用于采集图片。

问题已解决：从DLP PRO 4500数据手册的figure 3-7图可知，触发输出2，只在序列图开始时有个跳变；而触发输出1在每个图序中都有电平变化，可知使用触发1。（教训：认真看图，不认真想破头也不知道怎么回事；很多信息都在图上，要从这次的经历中吸取教训）
